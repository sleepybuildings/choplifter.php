# choplifter.php

A simple recreation of the classic [Choplifter](https://www.c64-wiki.com/wiki/Choplifter) game in PHP.

**This is a work in progress.**

## Requirements

- PHP 8.3
- Composer
- A terminal emulator

## Installation

```sh
composer install
```

## Run
```sh
./choplifter.php
```

## Controls

- Movement: `wasd`
- Quit game: `c`
